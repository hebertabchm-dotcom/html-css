Principais Formatações de Texto

Negrito / Destaque

Para deixar um texto em negrito, você pode usar a tag <strong> ou <b>.

Nessa frase temos um termo em negrito usando a tag <b> (não semântica)

Itálico / Ênfase

Para deixar um texto em itálico, você pode usar a tag <em> ou <i>.

Nessa frase temos um termo em itálico usando a tag <i> (não semântica)

Texto marcado

Para marcar um texto, você pode usar a tag <mark>.

Nessa frase temos um termo marcado usando a tag <mark>.

Texto grande e texto pequeno

Para deixar um texto em <big> (grande), você pode usar a tag <big>.

Nessa frase temos um termo grande usando a tag <big>.

Para deixar um texto em <small> (pequeno), você pode usar a tag <small>.

Nessa frase temos um termo pequeno usando a tag <small>.

Texto deletado

Para indicar um texto deletado, você pode usar a tag <del>.

Nessa frase temos um termo deletado usando a tag <del>.

Texto inserido

Para indicar um texto inserido, você pode usar a tag <ins>.

Nessa frase temos um termo inserido usando a tag <ins>.
