Html e Css

Html

Significado e Semântica

HTML (HyperText Markup Language) é a linguagem padrão usada para criar e estruturar páginas na web. A semântica em HTML refere-se ao uso de elementos que descrevem claramente o significado do conteúdo que eles envolvem. Elementos semânticos ajudam os navegadores, mecanismos de busca e tecnologias assistivas a entender melhor a estrutura e o propósito do conteúdo da página.

Exemplos de Elementos Semânticos em HTML:

  • <header>: Representa o cabeçalho de uma seção ou página.
  • <nav>: Define uma seção de navegação com links para outras páginas ou partes da página.
  • <main>: Indica o conteúdo principal da página.
  • <article>: Representa um conteúdo independente, como um artigo de blog ou notícia.
  • <section>: Define uma seção genérica de conteúdo, geralmente com um tema específico.
  • <aside>: Representa conteúdo tangencialmente relacionado ao conteúdo principal, como barras laterais ou anúncios.
  • <footer>: Define o rodapé de uma seção ou página.

CSS

Estilo, Forma e Layout

CSS (Cascading Style Sheets) é uma linguagem usada para descrever a apresentação de um documento HTML. Ele permite que os desenvolvedores controlem o layout, as cores, as fontes e outros aspectos visuais das páginas web. O CSS ajuda a separar o conteúdo (HTML) da apresentação (CSS), facilitando a manutenção e o design responsivo.

Exemplos de Propriedades Comuns em CSS:

  • color: Define a cor do texto.
  • background-color: Define a cor de fundo de um elemento.
  • font-size: Define o tamanho da fonte do texto.
  • margin: Controla o espaço fora das bordas de um elemento.
  • padding: Controla o espaço dentro das bordas de um elemento.
  • border: Define a borda ao redor de um elemento.
  • display: Controla como um elemento é exibido (por exemplo, block, inline, flex).
