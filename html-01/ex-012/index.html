Tags e HTML

As tags HTML são os elementos que compõem a estrutura de uma página web. Elas são usadas para definir o conteúdo e a apresentação da página.

Exemplos de Tags HTML

  • <h1> a <h6>: Definem títulos e subtítulos, sendo <h1> o mais importante e <h6> o menos importante.
  • <p>: Define um parágrafo de texto.
  • <a>: Cria um link para outra página ou recurso.
  • <img>: Insere uma imagem na página.
  • <div>: Define uma seção ou divisão na página, usada para agrupar elementos.
  • <span>: Usado para estilizar uma parte específica do texto dentro de um elemento.
