Oquê você irá aprender ?

Você irá aprender sobre as diferentes formas de representar cores no CSS.

  • Representação por nomes
  • Representação por códigos hexadecimais
  • Representação por códigos RGB
  • Representação por códigos HSL

Exemplo de Cores

Você pode representar as cores por nomes, como blue e white.

Exemplo de cores

Você pode usar a notação hexadecimal para definir cores. Por exemplo:

h1 {
    color: #ff0000;
}

Exemplo de Cores

Você pode usar a notação RGB para definir cores. Por exemplo:

h1 {
    color: rgb(255,0,0);
}

Exemplo de cores

Você pode fazer representação através de hsl. Por exemplo:

h1 {
    color: hsl(0, 100%, 50%);
}

Exemplo de cores

Você pode escrever o nome da cor e o Visual Studio quando você passar o mouse sobre o nome da cor, ele irá mostrar a representação em código.
